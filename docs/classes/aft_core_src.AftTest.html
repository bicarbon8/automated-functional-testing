Class AftTest

class to be used for executing some Functional Test Assertion after checking with any PolicyPlugin instances that have been loaded to confirm that the assertion should be executed based on referenced Test ID(s)

Ex:

await verify(async (v: Verifier) => {
  await v.reporter.info('doing some testing...');
  let feature = new FeatureObj();
  return await feature.returnExpectedValue();
}).withDescription('example usage for Verifier')
.and.withTestIds('C1234') // if PolicyPlugin.shouldRun('C1234') returns `false` the assertion is not run
.returns('expected value');

Param

the Func<Verifier, any> function to be executed by this Verifier

Returns

a new Verifier instance
